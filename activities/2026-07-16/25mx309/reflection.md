# Reflection

This task helped me understand the complete Git and GitHub workflow. I learned how to fork a repository, clone it to my computer, create a new folder, add a file, commit my changes, push them to my GitHub repository, and create a Pull Request.

The most challenging part was understanding the difference between the original repository and my forked repository. After practicing the steps, I became more confident in using GitHub for collaboration.

I spent about 2 hours completing the task, including understanding the workflow and creating the Pull Request.

Overall, this mission improved my confidence in using Git and GitHub and showed me how developers collaborate on shared projects.
