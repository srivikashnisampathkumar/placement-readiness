# Prompts Used

## Prompt 1

Design a JSON schema for a dynamic employee registration form that supports nested sections, validation rules, and conditional rendering.

---

## Prompt 2

Generate reusable React components capable of rendering different input fields dynamically based on a JSON schema.

---

## Prompt 3

Implement validation logic for dynamic forms supporting:

- Required fields
- Minimum age validation
- Email format validation
- Custom validation rules

---

## Prompt 4

Implement conditional rendering such that:

"If the user selects 'Employed', display the 'Company Name' field; otherwise hide it."

---

## Prompt 5

Design a scalable architecture for a JSON-driven form generator suitable for enterprise applications.

---

## Prompt 6

Explain the Big-O time and space complexity of rendering, validating, and evaluating conditional fields in a dynamic form engine.

---

## Prompt 7

Suggest improvements for making the dynamic form generator production-ready, including scalability, maintainability, and performance optimizations.
